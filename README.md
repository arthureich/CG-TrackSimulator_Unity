# Athletics Racing Simulator

## Description
Unity/C# athletics racing simulator developed for Computacao Grafica coursework in 2026. The project simulates track events with procedural skeleton runners, race-state logic, AI opponents, official lane/track generation, relay/cinematic modes, UI Toolkit screens, camera behavior, and race audio.

## Tech Stack
- Unity 3D
- C#
- UI Toolkit
- Procedural mesh/skeleton construction
- Race simulation logic
- Audio and camera systems

## Highlights
- Multiple race types such as 100m, 200m, 400m, 800m, 1500m, and 4x100m relay.
- AI racer configuration with reaction time and pacing ranges.
- Procedural skeleton builder and animator.
- Track/lane generation based on official-style dimensions.
- HUD, menu, results, camera-follow, cinematic replay, and audio managers.

## Structure
- `Assets/Scripts/` contains the custom gameplay and simulation scripts.
- `Assets/Scenes/` contains Unity scenes.
- `Assets/UI/` contains UI Toolkit layouts and styles.
- `Packages/` and `ProjectSettings/` contain Unity project configuration.

## How to Run
Open this folder with Unity and run the menu/game scenes.
